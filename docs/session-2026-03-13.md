## Lessons Learned

1. **HTML Entity Handling** - `strip_html()` needed to distinguish between:
   - Escaped entities (`&lt;div&gt;`) → should unescape to `<div>` (preserve tags)
   - Actual HTML tags (`<p>text</p>`) → should strip to `text`

2. **Test Isolation** - Use `@patch('app.TASKS_FOLDER', ...)` instead of relying on global state

3. **Multi-stage Dockerfiles** - Reduce image size by 60%+ while maintaining security (non-root user)

4. **CI/CD Best Practices**:
   - Test multiple Python versions
   - Include linting in CI
   - Add coverage reporting
   - Use Codecov for visibility
